# Financial Health Dashboard

## Project Overview

The Financial Health Dashboard is an interactive Power BI report designed to analyze an organization’s financial status. It provides insights into profitability, financial position, cash flow strength, and budgeting scenarios.

## Objective

To develop an interactive dashboard that analyzes an organization's financial health and supports budgeting and financial planning.

## Dataset

Dataset used: Financial Sheets Dataset from Kaggle.

The dataset contains financial information such as:
- Profit and loss data
- Balance sheet data
- Cash flow data
- Financial ratios
- Market-related metrics

## Dashboard Pages

1. Financial Overview
2. Income Statement
3. Balance Sheet
4. Cash Flow
5. Forecasting & Budget Planning

## Key KPIs

- Sales
- Net Profit
- Profit Margin
- Total Assets
- Debt
- Working Capital
- Operating Cash Flow
- Free Cash Flow
- Net Cash Flow
- Forecast Sales
- Forecast Net Profit
- Forecast Operating Cash Flow

## Power BI Features Used

- Power Query data transformation
- Unpivoting financial columns
- DAX measures
- KPI cards
- Matrix visuals
- Line charts
- Bar charts
- Scenario-based forecasting
- Conditional formatting
- Page navigation
- Synced slicers

## Business Value

The dashboard helps users understand financial health by identifying profitability trends, cash flow risks, debt position, and possible future performance under different growth scenarios.
