# CodeAlpha Power BI Internship Projects

This repository contains Power BI dashboard projects completed as part of the CodeAlpha Internship 2026. The projects focus on business intelligence, data visualization, KPI tracking, forecasting, and interactive dashboard development using publicly available datasets.

## Projects Included

### 1. Financial Health Dashboard

The Financial Health Dashboard analyzes an organization's financial status using profit and loss, balance sheet, cash flow, and forecasting indicators.

**Key features:**
- Income statement analysis
- Balance sheet analysis
- Cash flow analysis
- Profitability trend analysis
- Scenario-based financial forecasting
- Interactive slicers and page navigation
- Actionable financial insights for decision-making

### 2. Human Resources Analytics Dashboard

The Human Resources Analytics Dashboard supports HR decision-making by analyzing recruitment, turnover, satisfaction, engagement, performance, and future hiring needs.

**Key features:**
- Recruitment source analysis
- Employee turnover rate tracking
- Satisfaction and engagement analysis
- Performance analysis
- Workforce segmentation
- Scenario-based hiring needs forecasting
- Interactive HR decision-support visuals

## Dataset
The datasets used in this repository are publicly available datasets downloaded from Kaggle.

1. [Financial Sheets Dataset]([https://example.com](https://www.kaggle.com/datasets/pacificrm/financial-sheets))
2. [Human Resources Data Set, HRDataset_v14](https://www.kaggle.com/datasets/rhuebner/human-resources-data-set)

## How To Use

- Download or clone this repository.
- Open the .pbix files using Power BI Desktop.
- If Power BI asks for the dataset path, update the source path to the local data/ folder.
- Refresh the report.
- Use slicers and page navigation to explore the dashboard.
